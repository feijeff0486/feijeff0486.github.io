Front-matter

参数 描述
title 标题
date 建立日期（如：2019-12-10 14:34:36）
updated 更新日期
tags 标签（不适用于分页）
categories 分类（不适用于分页）
layout 布局
comments 开启文章评论功能
permalink 覆盖文章地址

分类和标签的区别：分类具有顺序性和层次性，而标签没有

layout

Hexo 有三种默认布局：post、page 和 draft，它们分别对应不同的路径，而您自定义的其他布局和 post 相同，都将储存到 source/_posts 文件夹

布局 路径 说明
post source/_posts 默认布局
page source 另起一页
draft source/_drafts 草稿
